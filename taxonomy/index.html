Descriptions Pricer Config et Static Data

  • Static Data : Caractéristiques permanentes d’un instrument (ISIN, MIC, currency, issuer LEI, calendrier, type de coupon, option, maturité…)
  • Pricing Config : Paramètres nécessaires à la valorisation (méthodes de calcul, courbes, conventions calendrier, volatilité…)
  • Pricer Template : Type de pricer QuantLib (Bond, Swap, Option, ABS, Mortgage…), choisi selon la nature/structure (voir templates).

Extensions QuantLib avec ORE (Open Risk Engine)

ORE (Open Risk Engine) enrichit QuantLib pour le pricing, le calcul de risque, la gestion des XVA, le reporting réglementaire, le support produits complexes et multi-portefeuilles.
Documentation : https://www.opensourcerisk.org/
Type de Pricer Extension ORE/QuantLib Capacités supplémentaires
BondPricer Multi courbes, conventions multiples, analytics de risque Courbes de spread, analyse de sensibilité, gestion du rating, simulation scénarios
SwapPricer Pricing multi courbes, collateralisation, XVA Valo OIS, CSA, CVA/DVA/FVA, reporting réglementaire EMIR/CRR
OptionPricer Modèles stochastiques étendus, calibration avancée Support exotiques, vol surfaces, arbre de corrélation, stress-test
FuturePricer Backtesting, chainage contrats, PnL attribution Gestion positions long/short, agrégation multi marchés
CreditDefaultSwapPricer Modélisation corrélations portefeuilles, XVA crédit Rapports CSAs, scénario risque, recovery rate multi source
FXPricer Multi devise, intégration courbes FX avancées Arbitrage FX, VaR multi paires, reporting stress
ABSPricer Moteur cash flow ORE, tranches, courbe crédit Rapport waterfall, triggers, pool timeline, simulations prépaiement
MortgagePricer Analytics mortgage, cashflow flags Préparamétisation PSA/CDR, fee servicing, Securitisation
CommodityPricer Asset modelling ORE, volatilities, correlations Reporting EMIR, structure contrats, stress valeur spot/future
ExoticPricer Simulation Monte Carlo ORE, modèles hybrides XVA exotiques, backtesting avancé, triggers/path-dependency
  • Les extensions ORE permettent une valorisation cross-asset, une analyse multi-portefeuille, la gestion de la collateralisation, du reporting (EMIR/CRR/IFRS9), et du risque de contrepartie (CVA/DVA/XVA).
  • ORE est compatible avec les templates QuantLib et permet l’intégration dans les workflows métier via API et reporting customisé.
