QuantLib Pricer Templates
How to configure pricers for all instrument types
BondPricer
Used for fixed-income securities: government, corporate, and structured bonds.
  • Set issue amount, coupon schedule, day count convention, and calendar.
  • Choose yield, discount, or clean price methods.
  • Configure amortization and optionality (call/put).
SwapPricer
Used for vanilla and exotic interest rate swaps.
  • Specify fixed and floating legs, spread, payment frequency.
  • Set up the curve and reference index (e.g., Euribor, Libor).
OptionPricer
Supports vanilla, barrier, Asian, and exotic options.
  • Input underlying, strike, expiry, style (European/American).
  • Configure volatility surface/model.
  • Select Black-Scholes, binomial, or Monte Carlo engine.
FuturePricer
For futures contracts across commodities, FX, or rates.
  • Specify contract, delivery date, margining scheme.
  • Choose settlement conventions.
CreditDefaultSwapPricer
For single-name or index CDS and tranches.
  • Select protection period, schedule, and recovery rate.
  • Configure ISDA setup and default probability curves.
FXPricer
Used for FX Vanilla, options, NDFs, and swaps.
  • Choose currency pair, notional, settlement conventions.
  • Set up volatility and forward curves.
ABSPricer
For asset-backed securities (ABS), including auto/credit card/collateralized pools.
  • Configure pool characteristics: asset type, distribution, prepayment assumptions.
  • Set tranching structure, waterfall rules, and credit enhancement.
  • Specify legal maturity and call/control features.
MortgagePricer
Used for mortgage-backed securities (MBS), including pass-throughs and CMOs.
  • Define mortgage pool: principal, coupon, payment schedule.
  • Model cash flows with prepayment/extension options (PSA/CDR curves).
  • Set up tranches, credit enhancement, and servicing fees.
